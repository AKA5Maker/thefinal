## 项目说明

这个项目包含两个应用：
1. 个人网站 - 展示个人信息和技能
2. 待办应用 - 一个简单的任务管理工具

两个应用都使用Docker容器化部署，通过Docker Compose进行管理。

## 部署步骤

1. 创建GitHub仓库
2. 开发个人网站
3. 集成待办应用
4. 配置Docker和Docker Compose
5. 设置GitHub Actions自动部署
6. 部署到服务器
